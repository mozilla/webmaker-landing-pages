Welcome to Webmaker!

Let us show you around by helping you to learn a little HTML and CSS, the official languages of the web.

Learn a bit of code by making your first webpage

It’s fun and easy. This page shows you how to make a simple web page in about two minutes or less. You’ll learn a little HTML, the language that makes up the web, along the way. It’s easier than you think! Thimble, Mozilla’s code editor, makes it easy.
